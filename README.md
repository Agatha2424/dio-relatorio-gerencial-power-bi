## Projeto: Relatório Gerencial de Vendas com Power BI
Este projeto consiste na criação de um relatório avançado no Power BI utilizando o conjunto de dados Sample Financials. O trabalho envolveu a construção de um dashboard profissional com layout estruturado, botões de navegação, segmentadores personalizados, indicadores e visuais interativos. A primeira página foi organizada com formas, cores e elementos que garantem identidade visual e boa navegação, incluindo botões com ícones e alternância de visuais no mesmo espaço. Foram inseridos gráficos como barras, linhas, área, treemap, matriz e cartões de KPI, todos configurados com dados de receita, lucro, categorias e períodos. A segunda página ampliou a análise com novos gráficos, filtros e botões de retorno, mantendo a navegação fluida. O relatório final foi publicado no Power BI Service, e o repositório contém o arquivo .pbix, imagens do desenvolvimento e referências utilizadas
